Development Changelog

📱 V3.1.0 (Mobile Ready Update)

  • Full Mobile Support: Every page on the platform has been fully optimized with responsive viewport scaling for smartphones and tablets.
  • Dynamic Layouts: Navigation bars, video grids, and creator channel layouts now stack beautifully on smaller screens for a seamless app-like experience.
  • Changelog Update: Added this V3.1.0 record.

🔥 V3.0.0 (The Platform Update)

  • UI Overhaul: The front page has been completely modernized with a Hero banner and a cleaner layout.
  • View Counters: Videos now track and display how many views they get.
  • Subscriptions & Sidebar: You can now subscribe to creators! A collapsible sidebar tracks all your subs globally.
  • Video Descriptions: Creators can now add descriptions to their uploads to provide context or links.
  • Top 3 Videos: The search page now proudly features the top 3 most-watched videos on the platform.
  • Account UI: Remade the profile picture upload UI to match the sleek drag-and-drop video UI (with a 5MB limit).
